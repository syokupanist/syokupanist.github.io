魔法のキャンバス

当時画風変換で流行っていたPrismaアプリを知って、

「深層学習による画風変換技術」と「お絵かき」を掛け合せたら面白そうというアイデアを抱きました。

画風変換のモデルを再実装しwebサービスとしてプロトタイピングした所、ある程度ものになったので、リリースしてみました。

コンセプト

お絵かき機能によって、アプリ内で絵を描くことができます。 自分で描いた絵を、いろんな画家の画風へと変化させられます。

自分の絵が有名な画家が描いたみたいになるなんて、まるで画家気分♪

複数の画家の画風をミックスしたり、画風を強くしたり、変換後の絵の上から自分で描き込めるといった機能があります。

簡単に本格的な絵を、自由に作り出すことができます。

試行錯誤しながら、自分だけのオリジナルアートワークを完成させましょう！

2016年9月~2017年2月頃(学部４年時)　実装

使用技術

  • Android
  • ruby on rails
  • flask
  • TensorFlow Serving
  • tensorflow実装による画風変換モデル
  • docker
  • VPS

モデル推論に必要なサーバープランの費用が高く、維持費がかかるため、公開を終了しました。
